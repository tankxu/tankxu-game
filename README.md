# Tank Xu Game

Personal game index for `game.tankxu.com`.

## Development

```bash
npm install
npm run dev
```

## Games

- `semi-trailer-parking-simulator`: playable 2D semi-trailer parking simulator.
- `excavator-sim`: in-development 3D excavator ball digging game.

Game links in `src/main.js` are relative deployment paths and can be adjusted when the games are published.
